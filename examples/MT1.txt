# Ejemplo de una MT
# Reconoce cadenas en el lenguaje L = { a^nb^m | m < n, n > 0 }
q0 q1 q2 q3  # conjunto Q
a b          # conjunto Σ
a b c .        # conjunto Γ
q0           # estado inicial
.            # símbolo blanco
q3           # conjunto F
q0 a q0 a R  # función de transición: δ (q0, a) = (q0, a, R)
q0 b q1 b L  # cada una de las transiciones en una línea distinta
q1 a q2 c R
q1 c q1 c L
q2 b q1 c L
q2 c q2 c R
q2 . q3 . S
